## Functionality

### Technologies used
- Canvas
- Web Audtio API
- Javascript
- HTML/CSS

### Keyboard

Users can use standard keyboard for controller.

Up, Right, Left, Down

Start(Any key)

### Personal Top Score

Users can see a record of their top scores via caching.

### Mute

Users can mute and unmute the sounds by pressing "m" or clicking the sound icon.
